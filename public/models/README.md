# 3D Models

Place `.obj` files (and their `.mtl` material files if applicable) in the appropriate subfolder.

## Structure

```
models/
  vehicles/    — Bulldozer, trucks, etc.
  buildings/   — Warehouse, gas station, market, etc.
  nature/      — Trees, rocks, bushes, etc.
```

## Naming convention

- Use lowercase kebab-case: `bulldozer.obj`, `gas-station.obj`, `pine-tree.obj`
- Place matching material files next to the `.obj`: `bulldozer.mtl`
- Place textures next to the model that uses them
